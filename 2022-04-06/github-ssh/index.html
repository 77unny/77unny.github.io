github ssh 등록 및 디렉토리별 계정 정보 등록하기

언젠가는 기록해야지 하면서 생고생했던 경험을 공유합니다.

회사 받은 장비 또는 개인 장비 세팅을 할때 마다 삽질을 했습니다.

github, ssh, 그리고 디렉토리별 github 계정 정보 변경해주기..

github 에서 21년 8월 13일 부터 ssh 권한으로 제한이 되었습니다.


해야할 순서

  1. ssh key 만들기
  2. ssh add
  3. github 연결
  4. config 설정
  5. .gitconfig 설정
ssh key 만들기

cd ~/.ssh 로 접근합니다.


ssh-keygen -t rsa -C "github email 주소" -f "ssh명"

개인 적으로 ssh 구분을 위해 f 옵션을 줍니다.

위와 같이 커맨드를 입력하게되면 사용할패스워드 등등 물어봅니다. (아무것도 입력안하면 패스워드 사용안함)

여기서 패스워드 입력하게되면 사용할때마다 패스워드 입력 해야하는..

그리고 key 크기를 입력 안하게되면 기본 값으로 됩니다. (기본3072 ~ 최대 4096)

ssh add

ssh-add test_ssh

아까 만들었었던 test_ssh 를 등록하는 과정입니다.

정상으로 등록되었다면 Identity added: test_ssh (test_github@gmail.com) 라고 뜹니다.

오류가 난다면 evel $(ssh-agent)

등록된 ssh 를 확인하고싶으면 ssh-add -l

저는 회사, 개인, 테스트용 이렇게 총 3개가 등록되어 있습니다.

github ssh 등록

github.com 접속하고 로그인 하신 후에 본인 아바타 누르신 후 settings 으로

SSH and GPG keys 에서 new SSH key 를 눌러서 ssh 등록을 해주세요.

여기서 등록 하실때는 public 인 test_ssh.pub 의 내용을 등록 하시면됩니다.

cat test_ssh.pub 을 치게되시면 내용이 나오는데 ssh-rsa ~ 본인 이메일까지 복사해서 등록해주세요.

config 등록 (여기서 이해가 중요함)

touch config 로 config 파일을 생성하세요. (vim or vi config 로 생성없이 바로 접근하셔도 됩니다.)

스크린샷의 등록용 처럼 작성해주시면 됩니다. (블로그 글 작성하면서 이해를 위한 예시 설명입니다.)

여기서 Host 에 작성한 부분은 주로 사용될 예정이니 편한걸로 작성해주세요

.gitconfig 등록 (디렉토리별 계정 정보 나누기)

touch ~/.gitconfig 로 config 파일을 생성하세요. (vim or vi ~/.gitconfig 로 생성없이 바로 접근하셔도 됩니다.)

디렉토리 구분을 위한 vim ~/.gitconfig-디렉토리 구분용

개인용, 회사용, 공부용 여러가지 나누기 위해 디렉토리를 별도로 설정해주세요. (디렉토리 나누실 이유없으면 global만 등록하고 스킵)

해당 디렉토리에서 구분이 잘되었는지 확인하시면됩니다~

clone 및 remote 설정하기

이제 clone 을 할때 ssh 로 클론 하게 됩니다.

여기서 git@github.com:baekCode/레포명.git 을 아까 세팅 해두었던

git@github.com-test:baeCode/레포명.git 으로 clone 받으시면됩니다! (디렉토리 구분안하셨으면 변경 없이 그냥 그대로 하시면 됩니다.)

디렉토리 구분을 했는데 이미 받은 git 라면??

레포를 다운 받은 디렉토리에서 git remote -v 확인해주세요.

origin git@github.com:baekCode/레포명.git (fetch)
origin git@github.com:baekCode/레포명.git (push)

git remote set-url origin git@github.com-test:baekCode/레포명.git

를 입력 하고 git remote -v 하여 잘 변경되었는지 확인해주세요.

origin git@github.com-test:baekCode/레포명.git (fetch)
origin git@github.com-test:baekCode/레포명.git (push)

그럼 이제 디렉토리별 자유롭게 github 계정을 사용 하시면 됩니다.
